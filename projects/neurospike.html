Overview

Investigating spiking neural networks (SNNs) and neuromorphic computing paradigms that bridge neuroscience and AI for energy-efficient, biologically-plausible learning systems. This research explores how biological principles can inspire next-generation AI architectures.

Research Areas

1. Spiking Neural Networks (SNNs)

  • Event-driven neural computation mimicking biological neurons
  • STDP (Spike-Timing-Dependent Plasticity) learning rules
  • Temporal dynamics of neural networks
  • Neuromorphic hardware considerations for edge deployment

2. Predictive Coding

  • Hierarchical predictive models inspired by cortical processing
  • Error minimization through prediction mechanisms
  • Bridging predictive coding with modern deep learning
  • Applications to continual and few-shot learning

3. Hybrid Architectures

  • Combining continuous and event-driven dynamics
  • Integrating traditional ANNs with SNNs
  • Adaptive, biologically-plausible learning rules
  • Low-power computation paradigms for sustainable AI

Technical Implementation

Brian2 Simulations

  • Implementing diverse neuron models (LIF, AdEx, Izhikevich)
  • Complex network dynamics and connectivity patterns
  • Synaptic plasticity rules (STDP, homeostasis)
  • Population-level behaviors and emergent dynamics

Differential Equation Solvers

  • Custom numerical methods for neural dynamics
  • Efficient simulation of large-scale networks
  • Analysis of stability and convergence properties
  • Parameter optimization and sensitivity analysis

Pygame Demonstrations

  • Visual representations of neural dynamics
  • Interactive demonstrations of learning processes
  • Real-time visualization of spiking activity
  • Educational tools for understanding SNN behavior

Key Concepts Explored

Temporal Attention Mechanisms

  • Time-aware processing in neural networks
  • Bridging neuroscience insights with AI architectures
  • Energy-efficient attention for sequential data

Continual Learning

  • Avoiding catastrophic forgetting in neural networks
  • Biological inspiration for lifelong learning systems
  • Few-shot adaptation mechanisms
  • Memory consolidation strategies inspired by sleep

Research Contributions

  • Studied predictive coding frameworks and their neural implementations
  • Implemented Brian2 algorithms for large-scale neural simulations
  • Developed educational demonstrations for SNN understanding
  • Explored hybrid SNN-ANN architectures for practical applications

Future Directions

  • FPGA implementation of SNNs for real-time processing
  • Neuromorphic hardware deployment (Intel Loihi, IBM TrueNorth)
  • Scaling SNNs to practical applications (vision, speech)
  • Integration with robotic systems for embodied learning
