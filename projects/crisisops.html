Problem
The Core Challenge

Standard RL environments simulate physics. CrisisOps simulates a rarer and harder problem: strategic human deception. In a software crisis scenario, some engineers over-report their progress to avoid accountability. The agent playing the PM role must cross-reference self-reports against observable ground-truth signals — commit activity, ticket ages, peer mentions — to detect who is lying before the project collapses.

This is framed as a counterfactual reward problem: the agent's score is its final project outcome minus what a naive greedy PM (who trusts all self-reports) would have achieved on the same episode. This forces the agent to learn to extract information value from its limited query budget, not just perform random due diligence.

Novel Mechanisms
6 New RL Design Contributions

Dynamic Candor Evolution

Caught liars become more honest mid-episode; unchecked liars grow bolder. Deception dynamics evolve based on the agent's intervention history, requiring adaptive strategies.

Social Testimony Graph

query_peer_opinion action lets the PM triangulate through peer-to-peer intel. Deceptive allies give coordinated alibis; the agent must break the alibi chain.

Political Capital Resource

Second earned resource (alongside action budget). Spend to compel truth (force_truth) or tip off a whistleblower. Introduces resource allocation tradeoffs beyond simple query optimization.

LLM-Powered Adversarial Agent

One team member per episode uses Ollama (qwen2.5:3b) with OpenAI fallback for adaptive deception. Rule-based liars are too easy — the LLM opponent generates coherent cover stories that must be actively probed.

Long-Horizon Memory Buffer

Episode history compressed every 8 steps and injected into observation. Agent must track deception patterns across 30 steps without losing context, testing temporal reasoning.

Adaptive Crisis Generator

EMA weakness tracking dynamically increases exposure to the agent's current blind spots — harder deception types appear more frequently as the agent gets better at easy cases.
