# Project NeuroSpike — Biologically-Inspired Neural Networks

**Timeline:** May 2025 – Present  
**Lab:** SenAI Lab, IIT Madras  
**Tech Stack:** Python, Brian2, Pygame, NumPy, Neuromorphic Computing

## Overview

Investigating spiking neural networks (SNNs) and neuromorphic computing paradigms that bridge neuroscience and AI for energy-efficient, biologically-plausible learning systems.

## Research Areas

### 1. **Spiking Neural Networks (SNNs)**
- Event-driven neural computation
- STDP (Spike-Timing-Dependent Plasticity) learning
- Temporal dynamics of neural networks
- Neuromorphic hardware considerations

### 2. **Predictive Coding**
- Hierarchical predictive models inspired by the brain
- Error minimization through prediction
- Bridging predictive coding with modern deep learning
- Applications to continual learning

### 3. **Hybrid Architectures**
- Combining continuous and event-driven dynamics
- Integrating ANNs with SNNs
- Adaptive, biologically-plausible learning rules
- Low-power computation paradigms

## Technical Implementation

### Brian2 Simulations
- Implementing neuron models (LIF, AdEx, Izhikevich)
- Network dynamics and connectivity patterns
- Synaptic plasticity rules
- Population-level behaviors

### Differential Equation Solvers
- Custom numerical methods for neural dynamics
- Efficient simulation of large-scale networks
- Analysis of stability and convergence
- Parameter optimization

### Pygame Demonstrations
- Visual representations of neural dynamics
- Interactive demonstrations of learning
- Real-time visualization of spiking activity
- Educational tools for understanding SNNs

## Key Concepts Explored

### Temporal Attention Mechanisms
- Time-aware processing in neural networks
- Bridging neuroscience insights with AI architectures
- Energy-efficient attention for sequential data

### Continual Learning
- Avoiding catastrophic forgetting
- Biological inspiration for lifelong learning
- Few-shot adaptation mechanisms
- Memory consolidation strategies
