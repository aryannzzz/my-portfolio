Overview

CamBrain is a sophisticated vision-language desktop agent that sees your screen, understands UI elements, plans multi-step tasks, and executes them autonomously. Built using the ReAct reasoning pattern with reflexion-based self-correction, it represents cutting-edge work in AI-powered automation.

🎯 Key Capabilities:
  • See your screen through real-time capture and OCR
  • Understand UI elements using vision encoders (SigLIP/CLIP)
  • Plan multi-step tasks using LLM reasoning (ReAct pattern)
  • Execute actions like clicking, typing, and scrolling
  • Learn from successes and failures through reflexion
  • Stay safe with built-in guardrails and rate limiting

Development Timeline

Week 1: Vision Pipeline
  • Screen Capture — Real-time capture at configurable FPS
  • OCR Extraction — PaddleOCR with bounding boxes
  • Vision Encoding — SigLIP/CLIP embeddings for scene understanding
  • Screen Parsing — Structured JSON representation of UI elements
Week 2: LLM Integration
  • Multi-Provider Support — Gemini, OpenAI, Ollama via LiteLLM
  • ReAct Agent — Thought → Action → Observation reasoning loop
  • Task Planning — Complex tasks decomposed into atomic actions
  • Prompt Templates — Extensible prompt engineering system
Week 3: Action Execution
  • Action Mapping — Natural language → screen coordinates
  • Action Verification — Before/after comparison for success check
  • Fuzzy Matching — Find UI elements even with typos
  • Retry Logic — Automatic retry on failure
Week 4: Memory & Learning
  • Reflexion Loop — Self-correcting agent with failure analysis
  • Memory Buffer — Short-term context (last N screen states)
  • Vector Memory — Long-term storage with ChromaDB
  • Preference Tracking — Learn user patterns over time
Week 5: API & UI
  • FastAPI Server — REST API for agent control
  • WebSocket Streaming — Real-time updates and logs
  • CLI Interface — Command-line access to all features
  • YAML Configuration — Flexible settings management
Week 6: Safety & Polish
  • Safety Guardrails — Block dangerous actions (rm -rf, format, etc.)
  • Rate Limiting — Prevent runaway automation
  • Rollback — Undo capabilities for reversible actions
  • Input Sanitization — Clean and validate action inputs

🛡️ Built-in Protection

  • Blocked Actions: rm -rf, format, shutdown, registry editing
  • High-Risk Confirmation: File deletion, email, financial transactions
  • Rate Limiting: Max 60 actions/min, 5 actions/sec
  • Supervised Mode: Requires human confirmation for actions
