Overview

Built a complete machine learning framework from scratch using only NumPy, achieving 99.9% accuracy parity with Scikit-learn while providing full mathematical transparency for educational purposes. Currently used by over 50 students to learn ML fundamentals.

Implemented Algorithms

1. Supervised Learning

Support Vector Machines (SVM)

  • Linear and kernel-based SVMs (RBF, Polynomial, Sigmoid)
  • SMO (Sequential Minimal Optimization) solver
  • Multi-class classification strategies (One-vs-Rest, One-vs-One)
  • Custom kernel functions for flexible decision boundaries

Random Forests

  • Decision tree implementation from scratch with CART algorithm
  • Bootstrap aggregating (bagging) for ensemble learning
  • Feature importance calculations using Gini impurity
  • Out-of-bag error estimation for model validation

2. Unsupervised Learning

K-Means Clustering

  • Lloyd's algorithm implementation with convergence guarantees
  • K-means++ initialization for better starting centroids
  • Elbow method for optimal K selection
  • Cluster analysis tools and visualization utilities

Principal Component Analysis (PCA)

  • Eigenvalue decomposition for dimensionality reduction
  • Singular Value Decomposition (SVD) approach
  • Explained variance analysis and scree plots
  • Data reconstruction and compression capabilities
